# widget

## Installation of Python Package

```sh
pip install widgets
```

## Development

To install the development dependencies, run:

```sh
pip install -e ".[dev]"
```

To develop the JavaScript code, run:

```sh
npm install
```

Prerequisites:

- Node.js
- npm

To run the development server, run:

```sh
npx vite
```

To build the JavaScript code, run:

```sh
npx vite build
```
